📖 وصف المشروع / Project Description
PressureLab هو تطبيق تعليمي تفاعلي متكامل يتيح للمتعلمين والطلاب والمهندسين استكشاف وفهم مبادئ ضغط المائع الساكن (Hydrostatic Pressure) وضغط الغاز (Gas Pressure) بطريقة عملية مرئية مباشرة.

يجمع المشروع بين المحاكاة الفيزيائية الدقيقة والرسوم المتحركة التفاعلية والذكاء الاصطناعي لتقديم تجربة تعليمية فريدة تشمل:

✅ حساب وتصوير ضغط المائع الساكن داخل حاوية مع تغيير نوع المائع والعمق
✅ محاكاة ضغط الماء في المباني السكنية وتحديد الحاجة لمضخات رافعة
✅ دراسة المعادلة البارومترية (Barometric Formula) وتأثير الارتفاع على ضغط الغاز
✅ مقارنة ستة غازات مختلفة عند ارتفاعات ودرجات حرارة متغيرة
✅ حاسبة قانون الغاز المثالي (Ideal Gas Law) التفاعلية
✅ عرض تفصيلي لأهم أجهزة قياس الضغط مع رسومات توضيحية
✅ نموذج ذكاء اصطناعي (Gradient Boosting) مدرب للتنبؤ بالضغط أثناء الحركة
✅ محول وحدات ضغط مدمج (Pa, kPa, MPa, atm, bar, mmHg)
🎯 المفاهيم الفيزيائية المغطاة
ضغط المائع الساكن / Hydrostatic Pressure
المفهوم	المعادلة	الوصف
الضغط الهيدروستاتيكي	P = ρ × g × h	الضغط الناتج عن وزن عمود المائع
الضغط الكلي	P_total = P₀ + ρgh	ضغط القياس + الضغط الجوي
قانون باسكال	P = F/A	الضغط يتساوى في جميع الاتجاهات
ضغط المبنى السكني	P_floor = P_supply - ρgh	تناقص الضغط مع الارتفاع في الطوابق
ضغط الغاز / Gas Pressure
المفهوم	المعادلة	الوصف
المعادلة البارومترية	P = P₀ × e^(-Mgh/RT)	تناقص الضغط مع الارتفاع أُسّياً
قانون الغاز المثالي	PV = nRT	العلاقة بين الضغط والحجم ودرجة الحرارة
تغير الجاذبية مع الارتفاع	g(h) = g₀ × (Rₑ/(Rₑ+h))²	تصحيح قيمة الجاذبية
تغير الكثافة مع الارتفاع	ρ = ρ₀ × e^(-Mgh/RT)	تناقص الكثافة مع الارتفاع

🤖 نموذج الذكاء الاصطناعي / AI Model
طريقة التدريب
توليد البيانات: تم إنشاء 8,000 نقطة بيانات باستخدام المعادلة البارومترية الفيزيائية الدقيقة مع إضافة ضوضاء عشوائية بسيطة
المعلمات المستخدمة: الارتفاع (Altitude) ، درجة الحرارة (Temperature) ، الكتلة المولية (Molar Mass) ، الضغط الأساسي (Base Pressure)
المتغير المستهدف: الضغط الناتج (Pressure) بالباسكال
الخوارزمية: Gradient Boosting Regressor (200 شجرة قرار)
التقسيم: 80% تدريب — 20% اختبار

التنبؤ بالحركة / Movement Prediction
